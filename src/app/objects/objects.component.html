All the methods belong to class Objects which are utility methods for operating on objects. Point to be remembered all methods are static. Let's see how we can use these methods.

How to use isNull and nonNull

  public static boolean isNull (Object obj)

The Method has very simple implementation. It is nothing but a null check and equivalent to obj==null. Method will return true if object is equal to null. 


  public static boolean nonNull (Object obj)

This Method is vice versa of the above one. It will return true if object!=null
