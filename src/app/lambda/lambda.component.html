Lambda -> Expression and syntax

The most important and known feature of Java 8 is Lambda. Lambda is a anonymous function. Lambda expression lets us to define a class with single method (abstract) in a very concise manner. Let's take an example to understand without wasting time.

Flyable.java 
  public interface Flyable {{oC}}
    public void fly();
  {{cC}}
Speakable.java 
  public interface Speakable {{oC}}
    public void speak(String birdName);
  {{cC}}

Flyable,Speakable are Functional Ineteface. An Inteface is called Functional Interface where we have only one abstract method to implement.

Bird.java 
  public class Bird {{oC}}

    public static void main(String[] args) {{oC}}
      
      //prior to Java se8
      Flyable oldBird=new Flyable() {{oC}}
        @Override
        public void fly() {{oC}}
          System.out.println("Old Bird is flying slow");				
        }
      };	
      oldBird.fly();
      
      //abstract method implementation with lambda with out parameter
      Flyable newBird=()->System.out.println("New Bird is flying fast");
      newBird.fly();
      
      //with parameter
      Speakable bird=(name)->System.out.println(name+" can speak.");
      bird.speak("Sparrow");
    }
  }
Output 
Old Bird is flying slow
New Bird is flying fast
Sparrow can speak.

Let's try to find out the meaning of lambda syntax. ()->

To understand better I am removing the sign -> and ( ) . So, the expression becomes ()fly{{oC}}}

In method signature we are not passing any argument that's why it's blank. To pass any argument, mention it in between of parenthesis. (name)->System.out.println(name+" can speak.");

We don't need to provide function name because Functional Interface has only one abstract method. So, Instead of name we write -> .

If you are writing code in Eclipse and hover over the pointer on -> , you can see

It -> is expecting a function with a String parameter.

Writing lambda expression is just opposite of regular method implementation. Regular Method: regMethod(value){{oC}}}, Lambda Expression: (value)->{{oC}}}
